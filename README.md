# Aurora Coinly API

Aurora Coinly es una aplicación para la gestión de finanzas personales, diseñada para ayudar a llevar el control de ingresos, gastos y presupuestos.

## Tecnologías
- .NET 9
- PostgreSQL
- MediatR
- Clean Architecture
- Entity Framework Core
- Swagger
